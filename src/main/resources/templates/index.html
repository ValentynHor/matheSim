This app demonstrates my proficiency in Spring Java, HTML-Thymeleaf, and a bit of JavaScript.

It is designed for students learning mathematics and includes three roles:

  1. Admin - has full control over the application.
  2. Teacher - can create tests/answers, assign tasks to students, manage students and their progress.
  3. Student - can complete tasks/tests, view results, and track their progress.

The app showcases various features such as user authentication, role-based access control, test creation, result tracking, and more.

It serves as a practical showcase of my skills in building web applications using the mentioned technologies.
